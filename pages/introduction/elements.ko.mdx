# 프롬프트 요소

프롬프트 엔지니어링으로 가능한 더 많은 예제와 어플리케이션을 다룰수록 프롬프트를 구성하는 특정 요소가 있음을 알게 될 것입니다.

프롬프트에는 다음 컴포넌트가 포함될 수 있습니다.

**명령** - 모델이 수행하기를 원하는 특정 작업 또는 지시

**컨텍스트** - 더 나은 응답을 위해 모델을 조종할 수 있는 외부 정보 또는 추가 컨텍스트를 포함할 수 있습니다.

**입력 데이터** - 응답을 찾고자 하는 입력 또는 질문입니다.

**출력 표시기** - 출력의 유형 또는 포맷을 나타냅니다.

프롬프트에 모든 컴포넌트가 필요한 것은 아니며 포맷은 해당 작업에 따라 다릅니다. 다음 가이드에서 보다 구체적인 예를 다룰 것입니다.
